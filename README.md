# Bright-Tv-Analysis
TV Viewers and User Analysis

# BrightTV Viewership Analytics Case Study
# Project Overview
This project analyzes BrightTV’s user profiles and viewer transaction data to generate actionable insights that support the company’s goal of growing its subscription base.

The analysis focuses on understanding user behavior, consumption trends, and content performance, and provides strategic recommendations for the Customer Value Management (CVM) team.

# Objectives
- Analyze user and usage trends
- Identify factors influencing content consumption
- Recommend content strategies for low-consumption periods
- Propose initiatives to grow the user base

# Dataset Description
The dataset includes:
- User Profiles (subscriber information)
- Viewer Transactions (session-level consumption data)
  
Key Notes:
- Each record represents one viewing session per subscriber
- Date and time are in UTC (converted to South African Time - SAST)
- Additional derived metrics were created to enhance insights
  
# Tools & Technologies
- Azure Databricks SQL – Data cleaning, transformation, and analysis
- Python – Data preprocessing and feature engineering
- Excel / Power BI / Google Looker Studio – Data visualization and dashboarding
- Loveable https://musiiwa.lovable.app/

  # Key Analysis Areas
1. User & Usage Trends
- Peak viewing days and times
- Session frequency and duration
- User segmentation (active vs inactive users)
  
2. Factors Influencing Consumption
- Time of day (weekday vs weekend patterns)
- Content type preferences
- User engagement levels
  
3. Low Consumption Insights
- Identification of low-traffic days
- Analysis of content gaps during those periods

  # Key Insights
- Consumption tends to peak during specific hours and days
- Certain content categories drive higher engagement
- Low-consumption periods present an opportunity for targeted content releases

# Recommendations
 # Content Strategy
- Introduce high-demand content during low-consumption periods
- Personalize recommendations based on user viewing history

  # User Growth Initiatives
- Launch targeted marketing campaigns
- Offer promotions or trial subscriptions
- Improve user onboarding experience

  # Engagement Optimization
- Implement recommendation engines
- Use push notifications to increase session frequency
  
  # Deliverables
- Data cleaning and transformation scripts (SQL/Python)
- Analytical insights
- Interactive dashboards (Power BI / Excel / Looker Studio)
- Strategic recommendations presentation (Microsoft Powerpoint)
